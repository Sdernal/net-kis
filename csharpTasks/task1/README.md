# Задание по .Net №1
В этом задании требуется реализовать простую структуру двумерного вектора. В файле **Vector.cs** находится заготовка, 
которую нужно допилить, в самом файле написано, что нужно сделать. 

Не забываем про документацию и тесты. Тестируем пока все в **Main**. Visual Studio поддерживает xml-документацию,
 поэтому попробуйте набрать "///" над методом и для вас любезно будет cгенерирован шаблон.
	
	/// <summary>
    /// Какой-то метод
    /// </summary>
    /// <param name="someParam">Какой-то параметр</param>
    /// <returns>42</returns>
    public int SomeMethod(int someParam) {
		return 42;
	}
		
	
 Как закончите с **Vector.cs** переходим в **VectorExtensions.cs**. И реализуем методы-расширения для вектора. 
Для того, чтобы создать метод-рамширение необходим статический класс - контейнер для статических методов. 
В методе первый агрумент указывается с ключевым словом **this** и далее у объектов этого типа можно вызывать данный метод:

	 public static class  ExtClass {
            public static void Print(this string someString)
        {
            Console.WriteLine(someString);
        }
    }

	string a = "Some String";
    a.Print();

Для сдачи задания нужно создать ветку от мастера, реализовать что нужно и сделать пулл-реквест.
 Для каждого задания нужно делать отдельную ветку, т.к. при успешно сданном задании пулл-реквест будет закрываться, 
а тащить все изменения в каждом задании как-то неоч. Ну чтож, развлекайтесь.
