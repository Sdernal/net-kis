# Задание по .Net №3
В этом задании вам предстоит поиграться с делегатами и событиями. Задание поделено на 2 части:
 ## 1. Биржа акции:
Тут надо реализовать простенькую биржу и брокеров, которые торгуют акциями. 
Суть такая: биржа берет и как-то выставляет цену той или иной акции и информирует
 брокеров и остальных подписавшихся. Подписавшиеся же покупают или продают эти акции. 
Надо собственно рализовать методы заглушки у биржи
 и у класса брокера, а также добавить еще один класс похожий на брокера. Например один из 
классов может только покупать, другой продавать, или один из классов покупает только один тип акций.
 В общем делайте как хотите, но должно получиться 2 класса с отличающейся логикой.

 Может показаться немного странной логика, но для того чтобы поиграться с событиями хватит. 
Если будут предложения, как структуризовать биржу лучше, пишите в комментариях к пулл-реквесту - обсудим.

## 2. Контейнер для персон
 
 Тут все проще, есть класс персона, есть для них контейнер. У контейнера нужно реализовать простенькие методы-заглушки.

Также вы могли заметить, что есть проект с тестами. Они должны проходить. Собственно тесты должны подсказать, какие аргументы нужно использовать в методах.
