# Задание по .Net №2

В этом задании требуется написать фабрику для всего движущегося.
Для этого у фабрики есть возможность добавления провайдеров (цехов) 
для создания того или иного транспортного средства. 

Использование фабрики:
```
Factory factory = new Factory(); // Простой конструктор
SomeProvider provider = new SomeProvider(options); // Создание провайдера, 
    // опции - аргументы конструктора
factory.AddProvider(provider); // Добавление провайдера в фабрику
factory.CreateSomething(); // Создание объекта с помощью провайдера
provider = factory.GetProvider<SomeProvider>(); // Явный запрос провайдера
provider.CreateObject(); // Создание объекта явно через провайдер
```  

Не забываем, что я грузовики - это тоже машины, т.е
```
factory.AddProvider(new TruckProvider());
Car car = factory.CreateTruck(); // Такое должно работать
```
Итак, нужно самим придумать иерархию классов, чтобы реализовать работу фабрики.
Как минимум фабрика должна уметь делать машины, грузовики и самолеты. 
Сами транспортные средства - просто классы с некоторыми свойствами: 
кол-во колес, крыльев, наименование и т.п. Также нужно заставить код в Program.cs работать.

## Подсказки:
- Открыв студию вы увидите недостающие файлы, возможно их названия подскажут вам идею возможной реализации
- Скорее всего вам понадобятся обобщенные интерфесы, а также ковариация для создания машин из цеха грузовиков.

## Формальные требования
- Код из Program.cs должен заработать
- Реализация создания минимум трех транспортных средств, два из которых связано наследованием
- Свой провайдер на кажое ТС
- Для каждого ТС методы-расширения фабрики.
